Pienimos

Pienimos is a domino clone application that utilizes the Google Maps API and the Google Geocoding API to provide users with an interactive map interface for ordering pizza. The application uses the geocoding API to convert user-entered addresses into geographic coordinates, which are then passed to the Maps API to display nearby Pienimo's stores on the map. The cart utilizes local storage, allowing users to keep track of orders that can be accessed and modified at any time. Overall, this application aims to provide an easy and user-friendly way for users to order pizza online.

Zonar

Zonar is a clone of the popular communication platform Discord that features live chat capabilities built with websockets. This allows for real-time communication and collaboration among users. In addition to live chat, the platform includes features such as customizable profiles, the ability to join servers and channels, and direct messaging. Utilizing websockets allows for a smooth and efficient user experience, making it a great tool for online communication and collaboration.

!Bnb

!Bnb is a vacation rental platform inspired by Airbnb. It has features like host profiles and utilizes AWS for efficient and dependable hosting. This platform is user-friendly for both property owners and travelers, whether they are searching for a unique vacation spot or looking to rent out their property.
